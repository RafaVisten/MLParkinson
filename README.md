# MLParkinson
Backup of scientific initiation on applying machine learning methods to aid in Parkinson's Disease diagnosis, which ran from 2024 .2 to 2025.2 and yielded two publications on the 2024 and 2025 ERI-GO (Escola Regional de Informática de Goiás) annals.

## Publications

[Parkinson’s Disease Classification Employing a Questionnaire of Non-motor Symptoms and Machine Learning Methods - Anais da Escola Regional de Informática de Goiás (ERI-GO) 2024](https://sol.sbc.org.br/index.php/erigo/article/view/32241)
[Identificação da Doença de Parkinson através da Análise de Dados Coletados por Smartwatch - Anais da Escola Regional de Informática de Goiás (ERI-GO) 2025](https://sol.sbc.org.br/index.php/erigo/article/view/39520)
